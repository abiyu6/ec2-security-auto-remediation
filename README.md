# EC2 Security Auto Remediation

## Description

This project demonstrates how to build a proactive security monitoring and automatic remediation system for Amazon EC2 instances using AWS services.

## AWS Services

- Amazon EC2
- IAM
- CloudWatch
- CloudTrail
- AWS Config
- EventBridge
- Lambda
- SNS
- Systems Manager
- GuardDuty
- Inspector
- Security Hub
- Terraform
